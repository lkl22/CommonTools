# 指令相关

## windows关闭java进程

> taskkill /F /IM java.exe 

## 指令打开文件

### Windows
打开host文件夹所在地址:
> start C:\WINDOWS\system32\drivers\etc

直接打开host文件:
> start notepad C:\WINDOWS\system32\drivers\etc\hosts

### macOS

> open filePath
